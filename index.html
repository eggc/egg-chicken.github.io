Grid System

Grid System とは、デザインを行う際に格子状の補助線を引いて、文字や図がその線に沿うように配置するデザインの手法。Web デザインの基本技術としてよく使われている。たとえば、Bootstrap や Meterial-UI は多くの機能を備えたライブラリだが、その基本機能として Grid System を利用できる。もし Grid System だけを利用したいなら、より軽量な Flexbox Grid のような選択肢もある。最新のブラウザでは CSS Grid Layout と呼ばれる機能が組み込まれたため、ライブラリ無しでも Grid System を利用できる。

各ドキュメントを読むだけで十分だが、古典的な文献に興味があるなら Grid Systems in Graphic Design がある。

Git

Git とは、ファイルのバージョン管理システム。体系的に学ぶには Pro Git を読む。

  • リモートリポジトリの URL を表示するには git remote -v
  • リモートリポジトリの URL を変更するには git remote set-url origin NEW_URL

TocBot

TocBot は Table of Contents を生成する vanilla JavaScript ライブラリ。使い方は下記の通り。

  1. ライブラリをインポートする
  2. コンテンツの h1 タグに id 属性を与える
  3. コンテンツの最上位要素に class="js-toc-content" 属性を与える
  4. table of contents を表示したい要素に class="js-toc" 属性を与える
  5. JavaScript で tocbot.init() を実行する

Emacs

Emacs はテキストエディタ。

拡張性が高く、設定次第でいろいろなことができるようになる。欠点は、使い方に慣れるまで長い時間がかかること。また、プラグインのアップデートや追加が遅いこと。自分でプラグインを開発したいなら Lisp を習得しなければならない。

GitHub Pages

GitHub Pages はウェブサイトのホスティングサービス。

ソースコードを GitHub のリポジトリに登録して、少しの設定をするだけで利用できる。お金はかからない。ホストが勝手に広告を出すこともない。個人利用ならほぼ自由に使えるが、暴力、エロ等は不可、転送量は 100GB/月 までと言った制限がある。

使い方を知るには Tutorial を読む。

cmus
